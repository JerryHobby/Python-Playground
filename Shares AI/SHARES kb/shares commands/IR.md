# IR - IGNORE AND REDISPLAY

## Description
The 'IR' command is used to ignore all changes and redisplay the current PNR.

**Warning: Using the 'IR' command will discard all unsaved changes.** 

## Usage
```
IR
```

## Example
```
IR
```
